
## About This Visualization
This chart displays how employees are distributed across different work groups in the Seattle Department of Construction and Inspections (SDCI). It allows us to easily see which departments have more or fewer staff.

## About the Data
The dataset contains 408 employee records. Each entry shows an employee and their assigned work group. The visualization highlights that departments like Land Use & Engineering Services and Inspections have the largest number of employees, while some groups have very few.
